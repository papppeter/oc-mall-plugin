# AddressList

The AddressList component displays a list of a user's addresses.

It generates edit links for the AddressForm component and gives the user the ability to delete an Address.

# Properties

There are no properties available for this component.

# Example implementations

TIP

This component is part of the MyAccount component. You do not have to implement it separately.
